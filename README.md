# Sistema de Autenticação de Usuários 🔐

Projeto desenvolvido para praticar conceitos de autenticação, cadastro de usuários e integração entre front-end e back-end utilizando JavaScript e Node.js.

## Tecnologias Utilizadas

* JavaScript
* Node.js
* HTML5
* JSON Server
* CSS3

## Funcionalidades

* Cadastro de usuários
* Armazenamento de informações em banco de dados JSON
* Tela de login
* Validação de acesso
* Comunicação entre interface e servidor

## Estrutura do Projeto

* `tela--de-login.html` → página de login
* `formulario.html` → cadastro de usuários
* `script.js` → lógica da aplicação
* `server.js` → configuração do servidor
* `db.json` → armazenamento dos dados
* `usuarios/` → arquivos relacionados ao gerenciamento de usuários
* `banco-de-dados/` → estrutura de persistência de dados

## Objetivo

Aplicar conceitos fundamentais de autenticação de usuários, manipulação de formulários, armazenamento de dados e comunicação entre cliente e servidor.

## Aprendizados

Durante o desenvolvimento deste projeto foram praticados:

* Manipulação do DOM
* Eventos JavaScript
* Validação de formulários
* Estruturação de aplicações Node.js
* Consumo e armazenamento de dados
* Organização de projetos web

## Como Executar

1. Instale as dependências:

```bash
npm install
```

2. Inicie o servidor:

```bash
node server.js
```

3. Abra os arquivos HTML no navegador.
